# Recipe-app
This Recipe App is a simple and functional web application designed to store, view, and manage cooking recipes in a clean and organized manner. It follows a minimal design philosophy, focusing on clarity and usability rather than unnecessary decoration.  The application allows users to browse recipes, view ingredients and preparation steps, and maintain a small personal collection of dishes. Each recipe is presented in a structured format, making it easy to follow while cooking.  Built using HTML, CSS, and JavaScript, the app relies on core web technologies instead of heavy frameworks. This keeps the project lightweight, easy to understand, and suitable for beginners who want to learn practical web development the traditional way.
